# iamport
Iamport Rest Api Modules can be installed as a composer.

https://github.com/iamport/iamport-rest-client-php

### composer
    "require": {
        "nanuly/iamport" : "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/ngyNanuly/iamport.git"
        }
    ],

### usage
  use \Nanuly\Iamport\Iamport;
